JS Unit Testing

Look in the Chrome developer console to see the output for default unit test. CTRL_SHIFT_J
You can try to test by using the following form.

Sample:

Command: max(99,1);
Expected: 99

Command: maxOfThree(10, 5, 90);
Expected: 90

Command: isVowel('A');
Expected: true

Command: sum([1,2,3,4]);
Expected: 10

Command: multiply([1,2,3,4]);
Expected: 24

Command: reverse("hello");
Expected: olleh

Command: findLongestWord(["a","hello","world","program"];
Expected: 7

Command: filterLongWords(["red", "green", "blue", "orange", "yellow", "white", "black", "purple", "pink"], 6);
Expected: orange,yellow,purple
